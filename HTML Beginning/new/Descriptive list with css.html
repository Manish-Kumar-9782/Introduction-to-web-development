Descriptive List

HTML
HypterText Markup Language: it is a simple markup based language in which we use markups/tags to wrap content to give them identification so can easily read, detect and manipulate them.
CSS
Cascading Style Sheet : it is used to put styles on HTML web page, we can set border, image, background-image, layout, font-size etc.
JavaScript
JavaScript is a programming language which is particualarly designed to work on the web browsers with HTML Elements. we can generate dynamic content, create, update and delete the HTML element and content.
