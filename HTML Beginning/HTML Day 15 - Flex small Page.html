HTML: HyperText Markup Language

HTML is a markup language which is used to make a meaningful and well defined structure of our content so we can share this content with the help of the internet over many other devices with the help of hyperlinks.

CSS: Cascading style sheet

CSS styles are used to set colors, basic box modal of any element, we can change font size, style, weight and alignment, we can also make a display layout of our page which can set elements on different position with a specified arrangement.

JavaScript: A Client side Programming Language

JavaScript is a general purpose programming language which is generally used to web page flow by controlling the behavior of html element and applying (rendering) the css effect, it can add or remove individual html element and also put css effect, we make our page dynamic by using the javascript.
