Non Inline Text Formatting element

b: bold :- b element is used to putting some weight dark color on the font.

strong: bold :- it works line b element but with more importent. it is used by screen reader, with more stresst on word.

i: italic :- i element is used to make italic (inclined text) font.

em: italic with importance :- It works like i element but it has more importance on the word, screen reader will use this to read for blid people, with more stress.

small: samll element is used to decrease the size of the text a little.

big: big element is used to increase the size of the text a little.

ins: ins element is used to show that content is inserted.

del: del element is used to show that, a text is removed.

mark: mark element is used to mark a text with color, it highlight the text.
