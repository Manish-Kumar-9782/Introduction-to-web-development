JavaScript HTML DOM Methods

HTML DOM

In a HTML DOM we have methods and propeties.

  • Methods: HTML DOM methods are used to perform some action on HTML elements.
  • Properties:HTML DOM properties are values of HTML elements that you can set or change.

The DOM Programming Interface

-> The HTML DOM can be ccessed with JavaScript and with other programming language as well.

-> In the DOM, all the HTML elements are defined as objects.

-> The programming interface is the properties and methods of each objects.

-> A property is a value that you can get or set like changing the content of an element

-> A method is an action you can dot like add or deleting an HTML element.

The innerHTML Property

The easiest way to get the content of an element is by using the innerHTML property.

The innerHTML property is useful for getting or replacing the content of HTML elements.
