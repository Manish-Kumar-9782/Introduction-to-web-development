The HTML DOM Document Object

The HTML DOM Document Object is the owner of all other objects in your web page.

-> The document object represents your web page.

-> If you want to access any element in an HTML page, you always start with accessing the document object.

Finding HTML Elements

We can access HTML elements by using the element method in this we will discussion about:

  • document.getElementById
  • document.getElementByTagName
  • document.getElementByClassName
document.getElementById:
→ If we have defined any element with a unique id then we can access that element by using getElementById method, with this we have a control over the element which has that unique id.
→ Note: if we give the two elements same id then the last element will override the first element with that id.
document.getElementByTagName:
→ Every element in HTML comes with a tag like div, form, table etc.
→ if we want to select all the items of that type of element, than we will use getElementByTagName method.
document.getElementByClassName:
→ If we have given HTML element some class name then we can access all the element binded to that class by using getElementByClassName method.

Changing HTML Elements

We can change the value/content/properties of the html element by using the properties of some element properties.

Here are we have some element properties:

  • element.innerHTML
  • element.attribute
  • element.property

we also have a method to set the attribute values.

  • element.setAttriubte(attribute, value)

Adding and Deleting Elements

We can add or delete an element by using document object in JavaScript.

Document object has some method for these operations.

  • docuemnt.createElement(element)
  • document.removeChild(element)
  • document.appendChild(element)
  • document.replaceChild(element)
  • document.write(text)

document.createElement:
→ In an HTML docuemnt, the document.createElement() method creates the HTML element specified by tagName, or an HTMLUnkonwnElement if tagName isn't recognized.
document.removeChild:
→ In an HTML document, the document.removeChild() is used to remove a child of an element from HTML document.
document.appendChild:
→ In an HTML document, the document.appendChild() method is used to add a new element at the last position inside the element.
document.replaceChild:
→ In an HTML document, the document.replaceChild() method is used to replace an old element from new one.
document.write:
→ In an HTML document, the document.write() method is used to write some content, element inside the HTML document.
Note:- we shoud avoid this method to write something after loading the html document, it will delete all the existed content and write new content.

Adding Events Handlers

We can make our event handlers by using the other of the HTML element like document.getElementById(id).onclick, we can pass a function which will perform some action after clicking on that element.
