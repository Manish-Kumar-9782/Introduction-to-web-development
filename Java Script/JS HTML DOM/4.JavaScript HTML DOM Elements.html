Finding HTML Element by id

The easiest way to find an HTML element in to DOM, is by using the element id.

→ If the element is found, the method will return the element as an object (in element.)

→ If the element is not found, element will contain null.

Finding HTML Element by tag Name

If we find the any element by using the tag Name then we will have all the element with that tag Name.

→ In this we will get all the h2 element.

Finding HTML Element by Class Name

If you want to find all HTML elements with same class name, use getElementsByClassName().

→ In this we will change the color of each h2 element with class "red".

Finding HTML Elements by CSS Selectors

If we want to find all HTML elements that match a specified CSS selector (id class, names,types, attributes, values of attributes, etc), use the querySelectorAll() method.

→ In this we will change the color of each h2 element with class "red".
