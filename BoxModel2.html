BoxModel characteritics

This is a box model representation, in which we see about the border, margin, padding properties.

  • Padding: padding is the internal space b/w content and border.
  • border: it is highlighted border b/w padding region and margin region.
  • margin: it is outer space around the border of the element.
  • ul: (unordered list) genreally it used to cotain the list item.
  • li:(list item) it a list item element which is used to contain content of the list.
