HTML Core Elements and Attributes

In every HTML Document we have four main elements that form the basic structure of the every HTML Document. these are html, head, title, body, HTML.

These four elements should appear in every HTML document to make the HTML basic structure.

The html Elements

The html element is the containing element for te HTML document. we should add this element just after declaring the document type declaration.

Each docuemnt should have the opening tag html and closing tag /html

In b/w the opening tag and closing tag we need to put the head and body elements.

Attribute Group

Attribute are placed on the opening tag of an element and provide extra information about the elemnt that carries them. All attributes consist of name and value; the name define a property of that element and the value is a value of that property.

→ For e.g xml:lang attributes describe the language used within that element.

  • Core Attributes: The class, id and title attributes.
  • Internationalization attributes: the dir, lang and xml:lang attributes.
  • UI Events: Attributes associated with events such as onclick, ondoubleclick, onmouseclick, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout, onkeypress, onkeydown and onkeyup.

Note: Together, the core attributes and the Internationalization attributes are known as universal attributes.
