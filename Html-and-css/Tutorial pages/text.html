Word wrap

Word wrap: 'word-wrap' is used to break the word when it goes to outside of the container.
By default it has the normal, but if we assign it by 'break-word' it will break the word as soon as word hit the boundry of the container

1.No word wrap

Hello this is manish kumar ManishkumarEngineeringStudent

2. word-wrap: break-word

Hello this is manish kumar ManishkumarEngineeringStudent

Note:- Break word works only if the length of the word is greater than the container.
