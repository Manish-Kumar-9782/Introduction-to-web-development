Progress bar

The simple progress bar

In this section we will made a simple progress bar, progress bar itself a span element with progress-bar class and div.meter is the container of the progressbar.

The background of the progress bar span element is set by using the css.

→ Here the steps to make the progress bar.

  1. First of all we need to set the position on the div.meter element as relative
  2. Now we need to set the some width of the div.meter as the width of progress-bar meter width.
  3. Set some background on the div.meter and we can add some more css properties to make other effects like border-color, border-style, border-width, box-shadow etc.
  4. Now we need to set the some css on the progress bar inner element which is span.
  5. set span.progress-bar position as absolute.
  6. set background color as we want.
  7. set left property as 0 or 1 for some padding effect
  8. Now we need to set some border radius on the span.progress-bar element
  9. Set some pre width for the progress-bar.

CandyStriping effect

we can get a cool striped effect by adding another element on top of span.meter element and laying a repeated css gradient over it. Semantically this is best achieved with a pseudo element.
