CSS

  • CSS: Cascading style sheet
  • Used to design the html web page
  • It can be applied inline, internally or externally.
  • Properties applied by using the Parent child relationship
  • File extension: .css

HTML

  • HTML: HyperText Markup language
  • It is used to make a structured format for a document which can be shared on multiple language.
  • HyperText: a text which can be shared b/w multiple computers using internet.
  • Markup Language: a language which is used to wrap the useful data or content by using some tag and elements.
  • It is used to make a meaningful structured web document which can be shared by internet and can be read any internet browser.
  • File extension: .html

Java Script

  • Java-Script: Java-Script is a client-side interpreted programming language
  • It is made and maintained by Sun MicroSystem (1995)
  • Java is used to make the page interactive and to enhance the page features, like forms, button, warning, pop up messages, notification, games, event controlling and more.
  • Initially it was made only work on the browser as a client-side but now a days after many changes and enhancement with some additional libraries, framework on runtime environment (node.js) we can use the Java-Script as the back-end language.
  • It is an Object-oriented, interpreted programming language.
  • it can be embed into the html document.
  • File extension: .js
