CSS 3D Transfrom Methods

The rotate3d(x,y,z,angle) method

The rotate3d method allows us to rotate the element in x,y and z-axis by specified angle

rotate3d

rotate(0.8,1,1,30deg)

The rotateX, rotateY and rotateZ method

The rotateX, rotateY and rotateZ method allow us to rotate an element along x, y and z-axis respectively.

rotateX

rotateX(120deg)

rotateY

rotateY(120deg)

rotateZ

rotateZ(120deg)

The translate3d() property

It is used to translate an element along x, y and z-axis

translate3d

translate3d(30px, 60px, 120px)

Note: here translate in z-axis is not visible we will see that when we will use the perspective property function.
