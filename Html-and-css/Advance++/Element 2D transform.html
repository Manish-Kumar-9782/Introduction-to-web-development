The translate() method

The tranlate() method moves an element from its current position (according to the parameter given for the X-axis and the Y-axis).

The rotate() method

The rotate() method rotates an element clockwise or counter-clockwise according to a given degree.

Move the mouse over the element to rotate the element.

rotate

60deg

Using the negative values will rotate the element counter-clockwise.

The transfom-origin property

The transform-origin property allows us to change the position of transformed elements.

transform-origin must be used together with the transfrom property.

by default the transform origin is the center of the element.
