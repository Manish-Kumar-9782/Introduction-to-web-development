CSS Layout Modes

A CSS layout mode, sometimes called layout, is an algorithm that determines the position and size of boxes on the way they interact with their sibling and ancestor boxes.

In this section we will learn:

  • Normal flow
  • Table layout
  • Float layout
  • Positioned layout
  • Multi-column layout
  • Flexible box layout
  • Grid layout

CSS Layout Modes

Normal flow

All elements are part of normal flow until we do something to tak them out of it. Normal flow includes block layout, designed for laying out boxes such as paragraphs and inline layout. which lays out inline items such as text.

Table layout

Designed for laying out tables. in this layout we configure the table data and arrange them in a row and column with some addition information.

Float Layout

Designed to cause an item to position itself left or right with the rest of the content in normal flow wrapping around.

Positioned Layout

Designed for Positioned elements without much interaction with other elements.

Multi-column Layout

designed for laying content out in columns as in a newspaper.

Flexible box Layout

Designed for laying out complex pages that can be resized smmothly

Grid Layout

Designed for laying out elements relative to a fixed grid.
