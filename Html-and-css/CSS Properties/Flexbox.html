Basic concepts of flexbox

the flexible box module, usually referred to as flexbox, was designed as a ont-dimensional layout model, and as a method that couyld offer space distribution between items in an interface and powerful alignment capabilities. This argicle gives an outline of the main features of flexbox, which we will be exploring in more detail in the rest of these guides.

When we describe felxbox as being one one dimensional we are describing the fact that flexbox deals with layout in one demension at a time-either as a row or a column. This can be contrasted with the two-dimensional model of CSS Grid Layout, which controls columns and rows together.

The two axes of flexbox

When working with flexbox we need to think in terms of two axes, the main axis and the cross axis. The main axis is defined by the flex-direction property, and the cross axis runs perpendicular to it. Everyting we do with flexbox refers back to these axes, so it we need to be carefull while using them.

The main axis

The main axis is defined by flex-direction, which has four possible values:

  • row
  • row-reverse
  • column
  • column-reverse

if we select row or row-reverse, our main axis will run along the row x-axis in the inline direction.

if we choose column or column-reverse and your main axis will run from the top of the page to the bottom in the block direction.

The cross axis

The corss axis runs perpendicular to the main axis, therefore if our flex-direction (main axis) is set to row or row-reverse the cross axis runs down the columns.

if our main axis is column or column-reverse then the cross axis runs along the rows.
