In this section we have used these elements.

  • header: a header tag to define the header of the document page.
  • nav: a navigation bar just below the the header bar.
  • aside: a aside or sidebar which may contain the other page related links.
  • section: a setion element which will contain the documents necessary text infromation.
  • footer: it is the last element which is lies below all the elements and contians copyright, author, contact, address etc. type information.
