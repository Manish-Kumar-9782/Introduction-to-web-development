1. The HTML title Element

The title element defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.

the title element is reuired in HTML documents!

The contents of a page title is very important for search engine optimization (SEO)! The page title is used by search engine algorithms to decide the order when listing pages in search result.

The title element:

  • defines a title in the browser toolbar
  • provides a title for the page when it is added to favorites.
  • dispalys a title for the apge in search engine-result

So, Try to make the title as accurate and meanuingful as possingle.

2. The HTML style Element

The style element is used to define style information for a single HTML page.

3. The HTML link Element

The link element defines the relationship b/w the current document and an external resource.

The linktag is most often used to link to external style sheets.

4. The HTML meta Element

The meta element is typically used to specify the character set, page description, keywords, autor of the document, and viewport settngs.

The metadata will not be displayed on the paage, but are used by browsers (how to disply content and reload page), by search engines (keywords), and other web services.

* Define the character set.

meta charset="UTF-8"

* Define keywords for search engines.

meta name="keywords" content="HTML, CSS, JavaScript"

* Define a description of your web page.

meta name="description" content="Free web Tutorials"

* Define the autor of a page.

meta name="author" content="Manish Kumar"

* Refresh document in every 30 seconds

meta http-equiv="refresh" content="30"

* Setting the view port to make our web page look good on all screen.

meta name="viewport" content="width=device=width, initial-scale=1.0"

Setting The ViewPort

The Viewport is the user's visible area of a web page. it varies with the device-it will be smaller on a mobile phone than on a computer screen.

meta name="viewport" content="width=device=width, initial-scale=1.0"

You should include the following meta element in alll your web pages.

This gives the browser instructions on how to control the page's dimensions and scaling.

The width=device-width parts sets the width of the page to follow the screen-width of the device (which will vary depending on the device).

The initial-scale=1.0 part sets initial zoom level when the page is first loaded by the browser.

5. The HTML script Element

The script element is used to define client-side JavaScripts.

6. The HTML baseElement.

The base element specifies the base url and/or target for all relative URLs in a page.

The base tag must have either an herf or target attribute present, or both.

The can only be one single base element in a document.
