Links and Navigation

In our daily life we have seen many document in desktop or lapton some of them are .docx, .txt which are simple text file bute if we see the .html file, they are also plain text files but in a structured format. which are used to put some content on the web page.

Now if we already have the .docx and .txt file then what is the need of the .html file, well the need is that a web page contain the hyperlinks which allows us to travels b/w the pages which are placed on different computer or network or Databases servers.

hyperlinks: Links or hyperlinks are the links which allows us to travel, to a different web page, which is placed at different location or different desktop/laptop/databse just by one click.

This ability of the web page make them different to any kind of the document format file, which is used to share the text decorated information by using the plain text file.

Internal Link: when a link is created and contain the it's own relative web page then that link is known as the internal link.

External Link: When a link is created an contain the external web site web page link then that link is known as the external link.

This section contains:

  • How to link b/w pages of a site.
  • How to link other site pages.
  • How to link to specific parts of a page in a site.

Basic Link

A link is specified using the a element. Anything between the opeaning a and the closing /a becomes the part of the link that users can click in a browser.

Link to a other web page

To link another web page, the opening tag a tag must define an attribute called herf; the value of the href attribute is the name of the file which need to be linked.

Note: We can put the absolute or relative path of the file, if file is stored in different folder or directory.

a href="Core Element and Attribute.html"Core element and Attribute of HTML/a

A link of the web page saved in current page folder. Core Element and Attribute of HTML

Link to another website page

To add a link of the other website page we just need to pass the absolute address as the value of href of that page which are we going to link.

Note: if we don't pass the current address of the page then browser will give us an error about not found the page.

a href="https://www.google.com"Google Home Page/a

A link of google home page: Google Home Page

Link as image

We can make an image as the link of another page to do that we just need to put the image-link in between the a element e.g.

a href="https://www.google.com"img src="google-128px.ico"/a

Link as image with title

we have seen that how to make an image a link by puttin the image in between the a element but it does not describe the anything about the link to the users.

we can add a short description or title of the web page which is also benificial for the screen readers.

This title will be shown as we put our mouse pointer/cursor on the image it will show the title of the page which will gives us the initial information about the linked page.

To add the title of the link which is in the form of the image, we just need to add the title attribute inside the opening tag of a.

a href="https://www.google.com" title="Google Home page"img src="google-128px.ico"/a

Linking to E-mail Address

We can convert a link to a mail address and we can send/recieve the mail to/from the given email address.

To add the email address we need to pass the email address with prefix mailto: as the value of href attribute.

a href="mailto:name@example.com"Email-Us/a Mail Me

Creating Bookmark Links with the a Element

Now we will see that how to make a link which will take us to a specified part of the current web page. The link which which is linked to a specifc part of the current web page is known as the bookmark link.

A bookmark link has two part Source and Destination of the link.

A bookmark link is very usefull when our page is very long and hard to find the desired content, then in that kind of condition we can make the bookmark link which will take us to the desired content if the link is defined at right point of destiantion.

Creating a source Anchor with the href Attribute

The source anchor is the link which allow us to click on the link and take us to the destination (a specified part of the page) link. A Source anchor can be in the form of text or image.

A source anchor does not contain any file path or link address as the value of the href attribute. it only takes the name of the id of the destination link with prefix #.

Syntax: a href="#idname"title/a

Syntax a id="basic_link"/a

The a Element's Other Attributes

The a element can carry several attributes some of them are used frequently or some of them are used very rarely. So in now we will see rest of the attributes.

Attribute: accesskey, charset, coords, href, hreflang, rel, rev, shape, style, tabindex, target, type

The accesskey Attribute

The accesskey attribute creates a keyboard shortcut that can be used to activate a link.

This shorcut key can be used to wake on the differnt section of the current page by defining the shortkey for every bookmark point.

We can also visit to a link by defining a shorcut key on that link defination.

A Simple example of the accesskey attribute

a id="#basic_link" accesskey="1"/a

use alt+1 to activate the Basic Link bookmark section.

Note: While using the shortcut key we should note that most of the browser use some predefined shortcut, so we should select the shortcut key without overriding the predefined shortcut keys.

Note: Note that the key is case-sensitive so we should use the value by knowning that here A and a both will have the different effect if we use them speperately.

The charset Attriubte

The charset attribute indicates the character encoding of the document, the URL points to. It is typically used only when we are linking to a page in a different language taht uses a different character set.

a href="https://www.amazon.co.jp" charset="ISO-2022-JP"Amazon Japan /a

Note: This attribute is useful when linking to forign language sites written in character encodings that some users might not be able to understand.

The coords Attributes

The coords attribute is designed for use on a source anchor when it contain an image. it allows us to create something known as image map, which defines the different parts of the same image link to different places.

The value of the coords attribute will be a set of x and y coordinates that indicates which part of the image should follow the link.

The hreflang Attribute

The hreflang attribute indicate which language the page we are linking to is written in. It is designed to be used when linking to a page in a different language from the current document, and the value of this attribute is a two-letter language code.

a href="https://www.amazon.co.jp" hreflang="JA"Amazon Japan /a

The rel Attribute

The rel attribute is used on the source anchor to indicate the relationship b/w the current document and the attribute, although it is possible that automated applications could be written to use this information.

The rev Attribute

The rev attribute provides the same role as the rel attributes but is used on the destination anchor to describe the relation b/w the destination and the source. it is currently not supported by major browsers.

The shape Attribute

if we want to create an image map, the shape attribute can be used to indicate the shape of an area that becomes a clickable hotspot.

The tabindex Attribute

By pressing the tab key we can change the focus of the interactive element. here tabindex plays an important role to specify that which element will take the focus first or which will take next.

The tabindex attribute allows us to specify the order in which order element will take the focus, when the tab key is pressed, the links obtain (or any interactive element) obtain focus.

When the user press the tab key, we may want to set the focus on a specific element (link/form).

The value of the tabindex attributes is a number b/w 0 to 32767.

The target Attribute

The target attribute is used to specify, that when the user click on a link, where the linked page sould open, by defalut it open in the current tab or window.

To open the linked page in other window or tab we need to set the target value as _blank

The title Attribute

The title attribute works here as we have discussed above while using the image as link.

It is used to give an additional information about the page or it can be used by the screen reader.

The type attribute

The type attibutes specifies the MIME type of the link. MIME types can be compared to file extensions, but are more universally accepted across different operating system.

For example: an html page would have the MIME type text/html , whereas a JPEG image would have the MIME type img/jpeg.
